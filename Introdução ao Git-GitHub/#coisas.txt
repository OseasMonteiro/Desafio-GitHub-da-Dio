O Git Bash é um terminal em que é possível manipular um projeto
em um repositório local.
Também é possível clonar um repositório no servidor 
remoto (GitHub) e manipulá-lo, modificando-o 
e atualizando seus status para empurrá-lo para o servidor remoto novamente.
